# European Retail Revenue Intelligence

## Problem Statement
Retail businesses require clear insights into revenue, sales trends, and regional performance to make data-driven decisions.

## Tools Used
- Excel
- MySQL
- Power BI

## Dataset
Retail sales dataset containing order details, revenue, regions, and product categories.

## Approach
- Data cleaning and preparation using Excel and SQL
- Data modeling in Power BI
- Created DAX measures for revenue, growth, and KPIs
- Designed an interactive dashboard for business users

## Key Insights
- Identified top-performing regions and products
- Seasonal revenue trends highlighted
- Revenue contribution by category analyzed
